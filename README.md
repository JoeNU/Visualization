# Visualization

A collection of code-based visualization experiments, motion graphics, and generative video tools.

## Projects

- [Neural Distillation Video](projects/neural-distillation-video/) - procedural Python/FFmpeg renderer for neon AI research explainer videos.

## Layout

Each project lives under `projects/` and keeps its own dependencies, README, and source code. Shared utilities can move into a top-level `packages/` or `shared/` directory later if multiple projects start using the same code.
